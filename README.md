# API-Rest-em-Node.js
Projeto básico com JavaScript em Backend. Utilizando Node.js e Express, focado em rotas REST, com consumo de API externa e dados em JSON.

API REST desenvolvida com Node.js e Express, responsável por consumir uma API externa de usuários e expor uma rota que retorna os dados de contato (nome e e-mail) em formato JSON.  

Projeto criado com foco no aprendizado de backend, rotas REST, async/await e consumo de APIs.

Como testar: 

1. Baixe o repositório
2. Instale as dependências
   npm install
3. Inicie o servidor
   node BackendServidor.js
4. Teste usando Postman ou Insomnia
